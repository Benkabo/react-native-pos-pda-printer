# react-native-pos-pda-printer

Enable printing for handheld terminal pda3505

## Installation

```sh
npm install react-native-pos-pda-printer
```

## Usage

```js
import { multiply } from 'react-native-pos-pda-printer';

// ...

const result = await multiply(3, 7);
```
